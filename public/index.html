Install Production Planning

Add this internal app to your home screen for standalone access.

iPhone or iPad Safari

  1. Open this Render HTTPS app link in Safari.
  2. Tap the Share button.
  3. Tap Add to Home Screen.
  4. Tap Add, then launch it from the new home-screen icon.

Android Chrome

  1. Open this Render HTTPS app link in Chrome.
  2. Tap the three-dot menu.
  3. Tap Install app or Add to Home screen.
  4. Confirm, then launch it from the app icon.

Desktop Chrome or Edge

  1. Open this Render HTTPS app link.
  2. Click the install icon in the address bar, or open the browser menu.
  3. Choose Install Production Planning.
  4. Launch it from the installed apps list or desktop shortcut.

This is a private internal PWA install. It does not use the App Store or Play Store, and live planner data still loads from the Render backend.
